MACHINE LEARNING
Machine Learning is a subset of AI that allows computers to learn from data and improve performance without being explicitly programmed.

types of Machine Learning
1.Supervised Learning
2.Unsupervised Learning
3.Reinforcement Learning

1.Supervised Learning
A type of learning where the model is trained using labeled data.
2.Unsupervised Learning
A type of learning where the model finds patterns in unlabeled data.
3.Reinforcement Learning 
Reinforcement Learning is a type of Machine Learning where an agent learns by interacting with an environment and improves its performance by receiving rewards or penalties.

algorithms
Linear Regression
Decision Tree
K-Nearest Neighbors
Support Vector Machine

DEEP LEARNING
Deep Learning is a subset of Machine Learning that uses neural networks with multiple layers to learn from large amounts of data.

algorithms
1. Artificial Neural Network (ANN)
2. Convolutional Neural Network (CNN)
3. Recurrent Neural Network (RNN)
